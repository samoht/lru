Module Lru_map.F

module F: sig .. end
F provides an immutable LRU map.

Average time complexity of single-element operations is O(log n).

Note The current implementation is backed by a simple non-balancing tree, and suffers from the O(n) pathological behavior these trees suffer from, if the mappings are inserted, or accessed, in key-order.
